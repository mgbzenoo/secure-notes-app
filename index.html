Secure Notes Web App

Write privately. Encrypt locally. Stay secure.
This sleek browser-based tool lets you write and store encrypted notes using your own password — no cloud, no account, no tracking.

  • AES-GCM encryption via Web Crypto API
  • Password-protected note saving
  • Local storage only — nothing leaves your browser
  • Dark mode interface

Use Cases: Personal journaling, storing sensitive ideas, practicing client-side encryption
