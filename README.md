# Secure Notes Web App

A privacy-first note-taking tool with local encryption using the Web Crypto API.

## Features
- Write and encrypt notes in your browser
- Password-protected using AES-GCM
- No cloud storage â€” everything stays local
- Dark mode UI

## How to Use
1. Write your note
2. Enter a password
3. Click "Encrypt & Save"
4. Later, enter the same password and click "Decrypt"

## Disclaimer
This is a local-only tool for personal use. It does not offer cloud sync or backup.
